Uso básico de la API de OpenAI

Este tutorial muestra cómo usar la API de OpenAI con la biblioteca oficial

Configuración Inicial

Importamos las librerías necesarias y cargamos las variables de entorno.

from openai import OpenAI
import os
from dotenv import load_dotenv

load_dotenv() # Carga el archivo de las variables de entorno

api_key = os.getenv('OPENAI_API_KEY')
client = OpenAI(api_key=api_key)

Solicitud a la API

Usamos el modelo gpt-4o-mini para enviar un mensaje y obtener una respuesta.

completion = client.chat.completions.create(
    model='gpt-4o-mini',
    messages=[
        {
            'role': 'user',
            'content': '¡Hola! ¿Me ayudas a aprender sobre la API de OpenAI'
        }
    ]
)

¡Hola! Claro que sí, estaré encantado de ayudarte a aprender sobre la API de OpenAI. La API de OpenAI permite a los desarrolladores integrar modelos de lenguaje, como GPT-3 y GPT-4, en sus aplicaciones. Aquí tienes un resumen de los aspectos más importantes:

1. Introducción a la API

  • La API de OpenAI te permite generar texto, responder preguntas, traducir idiomas, resumir contenido, y más, utilizando modelos de inteligencia artificial avanzados.

2. Registro y Autenticación

  • Para usar la API, primero debes crear una cuenta en el sitio web de OpenAI y obtener una clave de API.
  • La autenticación se realiza enviando tu clave de API en los encabezados de las solicitudes HTTP.

3. Puntos finales (Endpoints)

  • Completions: Para generar texto a partir de un texto inicial.
  • Chat Completions: Diseñado para la interacción de tipo chat, útil para construir asistentes conversacionales.
  • Edits: Para realizar ediciones o ajustes en un texto existente.

4. Realización de Solicitudes

  • Generalmente, haces una solicitud POST a la API con un cuerpo JSON que contiene los parámetros necesarios, como el modelo que deseas usar, el texto inicial, y otros configuraciones como temperature, max_tokens, etc.

5. Parámetros Comunes

  • model: El nombre del modelo que quieres usar (por ejemplo, “gpt-3.5-turbo”).
  • prompt: El texto inicial que alimentas al modelo.
  • max_tokens: El número máximo de tokens (palabras y fragmentos de palabras) que deseas obtener como respuesta.
  • temperature: Controla la aleatoriedad de las respuestas (valores más bajos generan respuestas más predecibles).
  • top_p: Usado para la muestreo en el núcleo, controla la diversidad de las respuestas.

6. Uso Responsable

  • Es importante seguir las pautas de uso responsable proporcionadas por OpenAI, para evitar generar contenido perjudicial o malintencionado.

7. Ejemplo de Código

Aquí tienes un simple ejemplo en Python utilizando requests:

import requests

api_key = 'TU_CLAVE_DE_API'
headers = {
    'Authorization': f'Bearer {api_key}',
    'Content-Type': 'application/json',
}

data = {
    'model': 'gpt-3.5-turbo',
    'messages': [{'role': 'user', 'content': '¿Qué es la inteligencia artificial?'}],
}

response = requests.post('https://api.openai.com/v1/chat/completions', headers=headers, json=data)

print(response.json())
